# Week 1: Terminal Supremacy

## Overview

Week 1 focused on building foundational Linux system administration skills through command-line operations, filesystem navigation, permission management, and log analysis. Through a series of hands-on labs, I learned to efficiently navigate headless Linux environments, secure sensitive files, automate administrative tasks, and extract actionable intelligence from system logs.

## Skills Developed

* Linux command-line navigation
* Filesystem Hierarchy Standard (FHS)
* File and directory management
* Linux permissions and ownership
* Read, Write, Execute (RWX) permissions
* Octal and symbolic permission notation
* Stream editing and text processing
* Log analysis using command pipelines
* Data extraction using grep, sed, and awk
* Security-focused troubleshooting

## Labs Completed

### Day 1: Filesystem Discovery & Navigation

Mapped the Linux Filesystem Hierarchy Standard (FHS), navigated restricted directories, located configuration files, and documented key system information.

**Artifact:** `discovery.txt`

### Day 2: System Hardening & Permissions

Audited a vulnerable Linux system, analyzed insecure permissions, and created an automated hardening script to remediate security risks.

**Artifact:** `harden.sh`

### Day 3: Threat Intelligence from Logs

Investigated web server logs to identify suspicious activity and extract attacker IP addresses using command-line pipelines and text-processing tools.

**Artifact:** `threat_ips.txt`

### Day 4: Incident Response Challenge

Combined navigation, permissions management, and log analysis skills to investigate a compromised server, repair broken permissions, and recover forensic evidence.

**Artifact:** `final_threat_report.txt`

## Key Takeaways

This week strengthened my ability to operate confidently in Linux environments while developing practical skills in system administration, permissions management, and command-line data analysis. These foundational competencies serve as the basis for secure system operations, incident response, and cybersecurity investigations.
