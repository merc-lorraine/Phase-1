# Week 3: Scripting the Defense

## Overview

Week 3 focused on using Python to automate security operations and transform manual administrative tasks into scalable security tools. Through scripting exercises, I learned how to analyze logs, identify threats, automate system audits, and generate structured security reports. The week emphasized using Python as a practical cybersecurity instrument for detection, monitoring, and incident response.

## Skills Developed

* Python fundamentals and data types
* Variable casting and input validation
* Python virtual environments (venv)
* Conditional logic and loops
* Lists and membership operators
* File input/output (I/O)
* Exception handling
* Log parsing and analysis
* Process monitoring and auditing
* JSON data generation
* Security automation and reporting

## Labs Completed

### Day 1: The Sentry (Python Foundations)

Built a Python-based port scanning utility using the socket library to identify active services on multiple systems. Learned to use virtual environments to isolate dependencies and create stable security tools.

**Artifact:** `port_check.py`

### Day 2: The Paper Trail

Developed a brute-force detection tool that scanned system logs, identified suspicious activity, and extracted attacker indicators from large datasets using loops, lists, and conditional logic.

**Artifact:** `brute_detector.py`

### Day 3: System Auditor

Created an automated process auditing tool that identified unauthorized processes and generated machine-readable alerts to support security monitoring and operational compliance.

**Artifact:** `system_auditor.py`

### Day 4: Operation Python Python

Completed a cumulative security automation challenge by building an incident response tool that analyzed security logs, identified brute-force attack activity, extracted attacker IP addresses, and generated a structured JSON threat intelligence report.

**Artifacts:**

* `incident_response.py`
* `threat_report.json`

## Key Takeaways

This week marked the transition from system operator to security tool builder. By combining Python programming fundamentals with cybersecurity workflows, I learned how to automate repetitive tasks, analyze security data at scale, and create reusable tools that support detection, monitoring, and incident response operations.
