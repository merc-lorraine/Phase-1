# Week 2: Network Foundations & Protocol Analysis

## Overview

Week 2 focused on networking fundamentals, including network interface management, IP addressing, subnetting, DNS resolution, and protocol analysis. Through hands-on labs and incident response scenarios, I learned how to troubleshoot connectivity issues, design network boundaries using CIDR notation, investigate services, and identify hidden communication channels.

## Skills Developed

* Network interface configuration and troubleshooting
* IP addressing and routing
* MAC address identification
* Default gateway configuration
* CIDR subnetting and binary calculations
* DNS analysis and remediation
* Service discovery and port enumeration
* Protocol interrogation and network diagnostics
* Incident response and network troubleshooting
* Linux networking tools (ip, ss, dig)

## Key Takeaways

This week strengthened my understanding of how modern networks communicate and how misconfigurations can disrupt operations. By working with routing tables, subnet masks, DNS services, and network protocols, I gained practical experience diagnosing connectivity problems, securing network infrastructure, and supporting incident response investigations.
